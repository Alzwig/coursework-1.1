# Курсовая работа по программированию за 1 курс 1 семестр. Тема: Обработка текста.
## **Аннотация**:
Консольное приложение на языке C, предназначенное для приёма многострочного текста через консоль, его первичной обработки и выполнения пользовательских операций. Программа предоставляет единый пользовательский интерфейс с выбором опций: подсчёт слов, реверс символов в словах, удаление предложений с запятыми, сортировки предложений по коду 5 символа предложения. Результат работы программы выводиться в консоль.

## **Сборка и запуск**:
### Для сборки проекта используется Makefile:
1. Перейдите в директорию с исходным кодом
```bash
cd src
```
2. Выполните сборку проекта
```bash
make
```
После успешной компиляции в каталоге появится исполняемый файл "cw"

3. Запуск программы
```bash
./cw
```
### Очистка проекта
Для удаления всех объектов компиляции используется команда:
```bash
make clean
```

## **Используемые технологии**:
- Язык: C
- Makefile для сборки

## **Обработка ошибок**
Все функции возвращают код ошибки из enum Error_Code.
При возникновении ошибки вызывается print_error(), выводящая сообщение вида:
Error: описание ошибки.

## **Особенности реализации**
- Нет использования break, continue, множественных return;
- Каждая функция имеет один выход;
- Выделение и освобождение памяти аккуратно организовано;
- Использованы структуры Sentence, Word, Text;
- Применена модульная архитектура.
